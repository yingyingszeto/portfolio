Product Onboarding

Customer Search | B2B2C

My Role

I was as a UX/UI Designer on maaiiConnect (now rebranded as Cinnox), a customer support platform via live chat on business websites.

Problem

To transform new user onboarding by making the LiveChat Widget installation accessible even for those without a technical background.

Problem

The original onboarding slides failed to properly instruct new users, leading to frequent requests for customer support. Users rarely give products a second chance if they find them difficult to use on the first try.

Outcome

[Text here...]

Final Design

Interactive step-by-step guide to set up the LiveChat service

The goal of a new user was to try the chatbox function by installing it. An easy-to-follow guide for them to install during onboarding is all they need while learning our product.

Using graphic instructions

Rather than the ambiguous illustrations and blocks of text used originally, using informational graphic instructions that are both intuitive and engaging.

Product tour for individual features

After the installation, users are provided with an interactive walkthrough that shows options to personalize the widget further. Just like having an in-person product demo.

Getting help without leaving the product

Users can quickly access the help center through a dedicated in-app window whenever they need support.
